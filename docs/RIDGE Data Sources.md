# RIDGE Data Sources

RIDGE relies on several external data sources.

## Terrain Data

Digital Elevation Models (DEM)

Used for:

- terrain funnel detection
- ridge identification
- saddle detection

## Satellite Imagery

Used for:

- habitat analysis
- vegetation detection
- bedding zone prediction

## Weather Data

Sources:

- NOAA
- OpenWeather
- WeatherKit

Used for:

- movement prediction
- wind modeling
- cold front detection

## Public Land Data

Sources:

- state GIS datasets
- federal land boundaries

Used for:

- public land mapping
- access point identification
