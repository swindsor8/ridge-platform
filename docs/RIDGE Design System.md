# RIDGE Design System

## UI Philosophy
The RIDGE interface should feel like a tactical hunting command center.

Characteristics:
- dark terrain maps
- high contrast overlays
- minimal clutter
- rugged outdoor aesthetic

## Navigation

## Map UI

Layers:
- satellite
- topo
- property boundaries
- public land

Markers:
- stands
- trail cameras
- travel routes
- bedding areas

Colors
